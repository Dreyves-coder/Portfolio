# 🚀 Portfolio – Dreyves Doukanda Mamboueni

Portfolio personnel développé en React, présentant mon parcours, mes compétences et mes projets.

## ✨ Sections

- **Hero** – Présentation avec effet de frappe animé
- **Compétences** – Stack technique avec barres de progression
- **Projets** – Réalisations personnelles et en entreprise
- **Expériences** – Timeline des stages et formations
- **Contact** – Formulaire de contact

## 🛠️ Technologies

- React 18
- CSS3 (animations, grid, flexbox)
- Google Fonts (Inter + Fira Code)

## 🚀 Lancer le projet

```bash
npm install
npm start
```

## 📦 Build pour la production

```bash
npm run build
```

## 🌐 Déploiement (Vercel)

```bash
npm install -g vercel
vercel
```
